# CLI Starter kit for PHP

## Introduction
Goal is to have minimal setup to start a new cli based projects.

## Features
- [x] Symfony Console
- [x] Symfony dependency injection
- [x] Symfony Config
- [x] Symfony var dumper


## Run
```bash
php main.php version
```

## Code Quality
```bash
## Run both phpstan and PHP CS Fixer
composer quality

composer analyse
composer lint
```
